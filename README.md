# mediasoup-poc

## Setup mediasoup-server

```
cd server
yarn install
yarn start-server
```

## Setup agora-client

1. Start Agora stream
2. Copy the APP_ID, TOKEN and CHANNEL values into agora-client/config.js

```
yarn start-agora-client
```

## Setup mediasoup-client

```
cd client
yarn install
yarn start
```
